# Data Structures & Algorithms

This folder contains DSA (Data Structures & Algorithms) implementations, practice problems, and solutions.

**Note:** This folder is completely separate from the Alien Invasion Game project.

## Folder Structure

- **arrays/** - Array-related problems and implementations
- **linked_lists/** - Linked list data structures and algorithms
- **trees/** - Binary trees, BST, and tree traversals
- **graphs/** - Graph algorithms and implementations
- **sorting/** - Sorting algorithms (quicksort, mergesort, heapsort, etc.)
- **dynamic_programming/** - DP problems and solutions
- **searching/** - Search algorithms (binary search, etc.)
- **stacks_queues/** - Stack and queue implementations

## How to Use

Add your DSA solutions and implementations to the appropriate folders. Each solution should include:
- Clear comments explaining the approach
- Time and space complexity analysis
- Test cases (if applicable)

Happy learning! 🚀
